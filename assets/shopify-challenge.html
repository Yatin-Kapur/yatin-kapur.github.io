Evidently, there is one shop which is charging just way more than the rest of the shops for the sneakers, and this is skewing the data in such a way which doesn't accurately represent the average order value. Now, to decrease the impact of the outlier, there are two options:

  1. take the median of all order_amount in the 'data' dataframe so that all data is accounted for, but outliers are largely ignored
  2. exclude this shop from the dataset and then take AOV using the same formula as before
